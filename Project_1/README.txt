# Project_ThaiOCR Folder Structure

## 1. Project_Management
- Proposal: ข้อเสนอโครงการ
- Timeline: ตารางงาน/แผนงาน
- Meeting_Notes: บันทึกการประชุม

## 2. Documentation
- Report_Draft: ไฟล์รายงานแต่ละบท (ร่าง)
- Final_Report: รายงานฉบับรวมและฉบับสมบูรณ์
- References: เอกสารอ้างอิง

## 3. Data
- Raw_Data: ข้อมูลดิบ เช่น รูปภาพ, PDF
- Processed_Data: ข้อมูลที่ทำ preprocessing แล้ว
- Annotation: ไฟล์ label/ground truth

## 4. Code_and_Experiments
- Source_Code: โค้ดหลัก
- Experiments: ไฟล์ทดลอง ผลลัพธ์
- Models: โมเดลที่ train

## 5. Presentation
- Slides: สไลด์นำเสนอ
- Demo_Video: วิดีโอสาธิตระบบ
- Images_and_Figures: รูปประกอบ/diagram

## 6. Submission
- Deliverables: ไฟล์ที่ต้องส่ง
- Final_Version: เวอร์ชันสุดท้าย ห้ามแก้ไข
